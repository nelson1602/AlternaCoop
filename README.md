# AlternaCoop

Plataforma para hostear la aplicación de la Cooperativa de Alterna Academy.

## Tecnologías

- **Frontend**: React con TypeScript
- **Backend**: Go

## Descripción

AlternaCoop es un proyecto destinado a servir como plataforma para la aplicación de la Cooperativa de Alterna Academy. Proporciona una solución moderna y escalable para gestionar las operaciones de la cooperativa.
